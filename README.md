# 2024 Olympic Data Analysis Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard analyzing Olympic Games data.  
The dashboard provides insights into medal distribution, country performance, and trends across different Olympic years.

## Key Insights
- Medal distribution by country
- Performance comparison across Olympic years
- Event-wise medal analysis
- Country participation trends

## Dashboard Features
- Interactive filters and slicers
- Medal type analysis (Gold, Silver, Bronze)
- Country-level comparison
- Year-wise Olympic performance

## Files in this Repository
- Olympic.pbix → Power BI dashboard file
- dataset.csv → Olympic dataset used for analysis
- screenshots/ → Dashboard preview images

## Tools Used
- Power BI
- Data Visualization
- Data Analysis

## How to Use
1. Download the `.pbix` file.
2. Open it using **Microsoft Power BI Desktop**.
3. Interact with the dashboard and filters to explore insights.
